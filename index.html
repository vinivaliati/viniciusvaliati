Regressão

Nesse Projeto, trabalhei com um conjunto de dados de casas na Califórnia, fornecido pelo scikit-learn, com o objetivo de prever preços de imóveis usando modelos de regressão. Utilizamos quatro diferentes modelos de regressão: Linear Simples, Linear Múltipla, Árvore de Decisão e SVR (Support Vector Regression). Cada modelo foi testado para encontrar o que melhor se ajustava aos dados.

Análise Exploratória

Na Análise Exploratória de dados, realizo uma investigação detalhada do conjunto de dados, buscando identificar padrões, tendências e outliers. Esse processo desempenha um papel fundamental na Ciência de Dados, pois orienta a formulação de hipóteses e as estratégias de modelagem subsequentes.

Data Cleaning

Data Cleaning é uma etapa de grande importância na Ciência de Dados, na qual lido com inconsistências, preencho ou removo dados ausentes, garantindo a confiabilidade e precisão dos modelos construídos a partir desses dados.

Feature Engineering

Feature Engineering é um processo fundamental na Ciência de Dados, em que eu crio e otimizo recursos partir de dados brutos, visando maximizar a eficácia dos modelos de aprendizado de máquina.

Machine Learning

Machine Learning é uma área da ciência de dados em que desenvolvo e aplico algoritmos que aprendem e se aprimoram com a experiência, permitindo previsões e decisões mais precisas com base em dados.

Story Telling
